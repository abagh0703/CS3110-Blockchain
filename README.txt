Running the code:
Preconditions, have a properly set up ocaml and opam environment and switch.

run "make install", this should install all of the neccessary dependencies if your environment is properly well. If not, then follow suggestions given by the error messages.

make test: will run our test cases by opening up a server and testing communicating with it.

make repl: this will run the repl, see the design doc for usage descriptions.
